# TCesium
TCesium是CesiumJS的一个实列库凌驾于原生实列的组合项目，作用于引导开发，和拓展开发
## CesiumJS
CesiumJS是一个JavaScript库，用于在web浏览器中创建三维球体和二维地图，无需插件。它使用WebGL进行硬件加速图形处理，并且是跨平台、跨浏览器的，并针对动态数据可视化进行了调整。
